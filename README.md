# monorepo-example

This repository is an example of a monorepo using [Bun](bun.sh), [Next.js](https://nextjs.org/), [Docker](https://www.docker.com/) and [Elysia](https://elysiajs.com)

To install dependencies:

```bash
bun install
```

To run:

```bash
bun dev
```

To run with docker compose:

```bash
bun docker:compose
```
